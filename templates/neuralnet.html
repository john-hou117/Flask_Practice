Convolutional Neural Networks: What are they?

• Type of neural network that has proven

successful in image recognition

and object classification.

• Also known as CNNs or ConvNets.

Why are ConvNets important? Where are they used?

Basic Building Blocks of ConvNets:

• Convolution

• Rectified Linear Unit (ReLU)

• Pooling

• Fully Connected Layer (Classification)

Convolution:

• Green = Original Image (5x5 matrix)

• Yellow = Filter (3x3 matrix), aka Feature Detector

• Red = Dot Product of Green and Yellow = Convolved Feature (aka Feature Map)

• Primary Purpose of Convolution = Detect Features (e.g. edges) in the image

Feature Map:

• A filter (box w/ red outline) produces one feature map

• Another filter (box w/ green outline) produces another feature map

Rectified Linear Unit (ReLU):

• After every convolution operation, an activation function, such as ReLU, is applied.

• All negative values become 0, while all values above 0 are unchanged.

• Primary Purpose of ReLU: introduce non-linearity (convolution is a linear process)

into our ConvNet because it must learn from non-linear data (images).

Pooling:

• Pooling reduces # of parameters in our network and controls overfitting

• Types of Pooling: Max, Average, Sum

• Primary Purpose of Pooling: Reduce Dimensionality

Fully Connected Layer:

• The fully connected layer means that every neuron in the

previous layer is connected to every neuron in the next layer.

• Activation Function: Softmax (basically, logistic regression for multiple categories)

• Primary Purpose of Fully Connected Layer: use features from convolution

and pooling layers for classifying the image. Our final result (since we used

softmax) will be output probabilities for each class.
